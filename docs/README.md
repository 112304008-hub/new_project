### 我是新手，應該從哪裏開始？

## 📝 文檔維護

### 更新頻率
- **架構文檔**：重大架構變更時更新
- **資料模型**：新增實體或欄位時更新
- **業務規則**：業務邏輯變更時更新
- **術語詞彙**：新增技術術語時更新
- **開發規範**：團隊共識變化時更新
- **常見問題**：發現新問題時立即添加

### 貢獻指南
1. 發現文檔錯誤或過時信息 → 提交 Issue
2. 補充新的常見問題 → 直接更新 `06_常見問題.md`
3. 添加新功能後 → 同步更新相關文檔
4. 文檔審查 → 遵循 Code Review 流程
